## 开发命令

### Installation

```bash
$ npm install
```

### Running the app

```bash
# 开发
$ npm run start:dev

# 构建
$ npm run build

# 更新 GEO IP 库数据
$ npm run updategeodb

# 生产环境运行
$ npm run start:prod
```

### Test

```bash
# 语法检查
$ npm run lint

# 测试
$ npm run test
$ npm run test:watch
```
